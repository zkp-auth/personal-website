# Personal Website
👋 Hi, I'm Nacir and I am a Fullstack Developer.


This is my personal website, built with Vite, React, and TypeScript.

This project will grow step by step as I design and build my online presence.

## Tech Stack

- Vite
- React
- TypeScript

## Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
